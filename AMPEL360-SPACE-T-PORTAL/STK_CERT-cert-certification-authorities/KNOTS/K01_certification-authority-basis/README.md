# K01: Certification Authority Basis

This directory contains the K01 knot workspace for certification authority basis activities.

## Quick Start

1. Read the master index: `00_00_IDX_LC01_SPACET_k01-authority-model-certification-basis_I01-R01.md`
2. Review task templates in the main K01 plan document
3. Check `MONITORING/` for current status

## Structure

- **ATA_TASKS/**: Per-ATA implementation tasks
- **DECISIONS/**: Authority decisions and agreements
- **EVIDENCE/**: Compliance evidence packages
- **MONITORING/**: Progress and NKU tracking
- **TEKNIA/**: External knowledge sharing
- **ASSETS/**: Supporting files
- **EXPORTS/**: Authority submission packages
- **ISSUES/**: Issue tracking
- **NOTES/**: Working notes

## Access

- **SPACET-INT**: Internal team access
- **SPACET-AUTH**: Authority pack-ready views

For questions, contact the Certification Manager or STK_CERT team.
